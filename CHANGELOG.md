# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2026-06-05
### Added
- Initial project structure with `README.md`, configurations parsing (`config.py`), database setup (`database.py`), Obsidian vault scanning logic (`process.py`), CLI command runner (`cli.py`), and automated build pipeline (`build.py`).
- Unit testing suite under `tests/test_kb_vault.py` covering vault detection, indexing triggers, config parsing, and file changes tracking.
